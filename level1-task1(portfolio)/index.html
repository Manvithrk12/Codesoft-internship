Workshop(Web development)

This workshop offers beginners a comprehensive introduction to website development over the span of 1-2 days. Covering HTML, CSS, and JavaScript fundamentals, participants learn to construct and style web pages while gaining insight into responsive design principles for cross-device compatibility. Additionally, they delve into version control using Git and GitHub, essential for collaborative coding projects. Through hands-on exercises and a final project, attendees not only grasp the basics of web development but also cultivate practical skills in building their own functional websites. By the workshop's conclusion, participants emerge equipped with foundational knowledge and resources to continue their journey in web development independently.
